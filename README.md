# PPO for Autonomous Highway Driving 🚗

## Complete Guide → [📖 README_COMPLETE.md](README_COMPLETE.md)

This project demonstrates **Proximal Policy Optimization (PPO)** applied to autonomous highway driving.

### ⚡ Quick Start
```bash
python train_ppo.py      # Train PPO agent
python train_a2c.py      # Train A2C for comparison
python compare_agents.py # Direct comparison
python plot_results.py compare  # Generate plots
```

**For detailed guide, project overview, configuration, and all instructions → [👉 Read README_COMPLETE.md](README_COMPLETE.md)**

### 🎯 What You'll Learn
- PPO algorithm (actor-critic, clipping mechanism)
- Comparison with A2C algorithm
- RL concepts: GAE, policy gradients, advantage estimation
- Training autonomous driving agents

### 📊 Expected Results
- PPO: ~35 mean reward (stable)
- A2C: ~28 mean reward (erratic)
- Improvement: 5-6x better than random agent
