\section{Introduction}
Reinforcement Learning (RL) has emerged as a powerful paradigm for training autonomous agents in complex decision-making environments. Unlike supervised learning, RL agents learn through direct interaction --- receiving rewards for desirable actions and penalties for mistakes.

In this project, we apply \textbf{Proximal Policy Optimization (PPO)}---a state-of-the-art actor-critic algorithm---to \textbf{autonomous highway driving}. The agent must navigate through multi-lane traffic, avoid collisions, and make intelligent lane-change decisions to overtake slower vehicles using the \texttt{highway-fast-v0} environment. To demonstrate a foundational understanding of Reinforcement Learning mathematics, both the PPO and \textbf{Advantage Actor-Critic (A2C)} algorithms were implemented \textbf{completely from scratch using PyTorch}, specifically avoiding high-level ready-made libraries. This allowed for a direct comparison of PPO's clipping mechanism against the standard A2C baseline.

\subsection{Problem Statement}
Train an RL agent to: (1) Maximize speed for progress, (2) Avoid all collisions, (3) Execute safe overtakes, and (4) Maintain stable driving behavior.

\subsection{Environment: Highway-fast-v0}
The environment simulates a 5-lane highway with multi-agent interaction and realistic physics.

\section{Methodology}
\subsection{Actor-Critic Architecture}
We use an actor-critic framework with shared hidden layers (256x256 MLP). The \textbf{Actor ($\pi_\theta$)} decides the action, while the \textbf{Critic ($V_\phi$)} evaluates the state. 
\begin{equation}
    \text{Input}(s) \rightarrow \text{FC}(256) \rightarrow \text{FC}(256) \rightarrow [ \pi_\theta(a|s), V_\phi(s) ]
\end{equation}

\subsection{Generalized Advantage Estimation (GAE) \& PPO}
We use GAE ($\gamma=0.99, \lambda=0.95$) to balance bias and variance. PPO prevents destructive policy updates using a \textbf{clipped surrogate objective}:
\begin{equation}
    L^{\text{CLIP}}(\theta) = \mathbb{E}_t \left[ \min\left( r_t\hat{A}_t,\; \text{clip}(r_t, 0.8, 1.2)\hat{A}_t \right) \right]
\end{equation}
The total loss $L = L^{\text{CLIP}} - 0.5 L^{\text{VF}} + 0.01 H[\pi_\theta]$ includes value loss and entropy bonus for exploration. Building these custom PyTorch engines allowed direct insight into the differences between PPO's clipped surrogate objective and A2C's standard policy gradient, providing a robust modular framework for autonomous driving.

\section{Implementation \& Results}
Training used an NVIDIA RTX 3050. Both custom PyTorch agents (PPO and A2C) were trained for 50k timesteps with optimized learning rates ($3\text{e-}4$ for PPO and $7\text{e-}4$ for A2C).

PPO demonstrates smooth improvement with clear convergence. Our from-scratch A2C implementation, while functional, exhibited significant variance and instability compared to PPO, highlighting the critical importance of the clipping mechanism in high-dimensional state spaces.

\subsection{Key Findings}
\begin{itemize}
    \item \textbf{Safety:} Our Custom PPO achieved a highly resilient 4\% crash rate in dense traffic, vastly outperforming both Custom A2C and the random baseline which both failed to avoid collisions effectively.
    \item \textbf{Stability:} PPO's clipping prevented the destructive policy updates observed in our A2C implementation, leading to much more reliable convergence.
    \item \textbf{Behavior:} The PPO agent learned to weave through traffic and execute high-speed overtaking maneuvers safely.
\end{itemize}

\section{Conclusion}
Building both PPO and A2C from scratch in PyTorch conclusively proved PPO's superiority in stability and safety for autonomous driving. The customized implementation provided depth to the analysis of clipped objectives versus standard policy gradients in complex traffic environments.
