\section{Introduction}
Reinforcement Learning (RL) has emerged as a powerful paradigm for training autonomous agents in complex decision-making environments. Unlike supervised learning, RL agents learn through direct interaction --- receiving rewards for desirable actions and penalties for mistakes.

In this project, we apply \textbf{Proximal Policy Optimization (PPO)}---a state-of-the-art actor-critic algorithm---to \textbf{autonomous highway driving}. The agent must navigate through multi-lane traffic, avoid collisions, and make intelligent lane-change decisions to overtake slower vehicles using the \texttt{highway-fast-v0} environment. We additionally train an \textbf{Advantage Actor-Critic (A2C)} agent to perform a comparative analysis showing why PPO's clipping mechanism leads to superior stability.

\subsection{Problem Statement}
Train an RL agent to: (1) Maximize speed for progress, (2) Avoid all collisions, (3) Execute safe overtakes, and (4) Maintain stable driving behavior.

\subsection{Environment: Highway-fast-v0}
The environment simulates a 5-lane highway with multi-agent interaction and realistic physics.

\section{Methodology}
\subsection{Actor-Critic Architecture}
We use an actor-critic framework with shared hidden layers (256x256 MLP). The \textbf{Actor ($\pi_\theta$)} decides the action, while the \textbf{Critic ($V_\phi$)} evaluates the state. 
\begin{equation}
    \text{Input}(s) \rightarrow \text{FC}(256) \rightarrow \text{FC}(256) \rightarrow [ \pi_\theta(a|s), V_\phi(s) ]
\end{equation}

\subsection{Generalized Advantage Estimation (GAE) \& PPO}
We use GAE ($\gamma=0.99, \lambda=0.95$) to balance bias and variance. PPO prevents destructive policy updates using a \textbf{clipped surrogate objective}:
\begin{equation}
    L^{\text{CLIP}}(\theta) = \mathbb{E}_t \left[ \min\left( r_t\hat{A}_t,\; \text{clip}(r_t, 0.8, 1.2)\hat{A}_t \right) \right]
\end{equation}
The total loss $L = L^{\text{CLIP}} - 0.5 L^{\text{VF}} + 0.01 H[\pi_\theta]$ includes value loss and entropy bonus for exploration. A2C serves as our baseline, lacking this clipping mechanism.

\section{Implementation \& Results}
Training used an NVIDIA RTX 3050. Agents trained for 50k timesteps with a learning rate of $3\text{e-}4$ (PPO) and $1\text{e-}3$ (A2C).

\subsection{Key Findings}
\begin{itemize}
    \item \textbf{Safety:} PPO achieved a 0\% crash rate, superior to A2C's 2\%.
    \item \textbf{Stability:} PPO's standard deviation (0.89) was 3$\times$ lower than A2C's (2.67).
    \item \textbf{Behavior:} The agent learned to weave through traffic and overtake safely.
\end{itemize}

\section{Conclusion}
PPO conclusively outperforms A2C in stability and safety for autonomous driving. The clipped objective function allows for reliable learning in high-dimensional traffic environments.
